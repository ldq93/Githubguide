Góc phải bên dưới có bash(powerShell), bấm mũi tên xuống chọn select default - chỗ command phía trên chọn chế độ Git bash


xuống bấm thùng rác ở terminal rồi mở lại terminal sẽ ra chế độ Git bash

Kiểm tra trạng thái của Git: dưới terminal bấm git status

gõ "git init" để khởi tạo git, sau khi tạo thì trong folder chứa file index sẽ có file ẩn git

sau khi kiểm tra git status thì untracked files present,

cần add thư mục dự án vào staging area, "git add ." để add tất cả thư mục, "git add tên_thư_mục" để add đúng thư mục cần

"git commit" để đẩy code lên github

copy paste "git commit -m "first commit"" và chạy thì báo ko nhận dạng được email

copy đoạn thông báo "git config --global user.email + "email" (đăng ký github) " rồi chạy

và copy đoạn thông báo "git config --global user.name + "tên user" " và chạy

chạy lệnh "git branch -M main" để thay đổi nhánh làm việc từ master đang chạy ở github thành nhánh main trên github

chạy "git remote add origin https://github.com/ldq93/Githubguide.git" để mở cổng kết nối đến github

chạy "git push -u origin main" để push code lên github" hiện bảng thông báo bấm "sign in with a code"

tiếp tục hiện ra bảng, copy đoạn mã rồi bấm link dòng dưới rồi paste đoạn mã vào đấy

bấm "continue -> authorize git-eco"

hướng dẫn push code lên git sau khi sửa
